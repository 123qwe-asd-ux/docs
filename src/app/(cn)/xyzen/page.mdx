# Xyzen

Xyzen 是 ScienceOL 矩阵中的 AI 担当。

<Note>
Xyzen 提供三种部署方式：

1. 通过 Docker 部署
2. 通过 源码部署
3. 前端通过 npm 包方式集成在你的已有项目中
</Note>

## Features

### 1. 无需全栈部署、快速集成到你的网站中

Xyzen 提供全栈开源，前端以 NPM 包方式分发，你可以一键快速的集成在你的 React 前端项目中。

后端提供一键部署，可用于你的前端连接。

### 2. 一键创建 MCP 服务器，并与各 AI 提供商集成

Xyzen 支持通过一键部署 MCP（Model Context Protocol）服务器，轻松连接主流 AI 服务商（如 OpenAI、阿里云、百度、智谱等），实现模型统一管理和智能路由。你可以根据业务需求，灵活切换和组合不同 AI 能力，提升系统的智能化水平和扩展性。

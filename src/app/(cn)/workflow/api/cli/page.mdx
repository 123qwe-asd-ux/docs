# CLI

Protium 提供 python package 和命令行工具用于简化你的自动化流程，要使用命令行工具，请使用 pip 安装。


## 安装

```bash  {{ title: "Terminal"}}
pip install -U protium
```

<Note>
如果你对 Python 及 pip 尚不了解，请先移步参考文档：[《了解 Python 与 Pip》](#) 
</Note>

以下提供一些简单的上手示例，帮助你构建自动化过程。

## 在命令行中使用 protium

### 查看所有的工作流

```bash
ptm list 
```

如果你还没有登陆，ptm 工具包会首先要求你输入 Personal AccessToken，AccessToken 的获取方式请见

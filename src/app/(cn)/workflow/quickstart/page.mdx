# Quickstart

这个页面用来帮助你在 Protium Workshop 中快速开始。

<Note>
  在正式开始运行之前，你将需要先从设置中获得你的计算资源 [访问令牌 (Access Token)](#)。
</Note>

## Create your first Node

进入 <a href={WorkflowSite}>Workspace</a>，点击左侧边栏或画布按钮创建第一个工作流。

<Img src="/workflow/quickstart/1.png" />

点击左上角的标题可以进行改名，右键点击画布，选择你要创建的节点，点击创建。

<Img src="/workflow/quickstart/2.png" />

使用相同的操作，依次创建 VASP 计算所需的 `POSCAR`、`KPOINTS`、`INCAR` 文件，以及 `Solver -> VASP` 求解器。

## Connect

拖拽每个节点中的 Handle，你将可以为你的节点创建连接关系。允许连接的节点将会以高亮进行展示。
这类允许拖动的部分在 Protium 中被称为 Handle，查看一个完整的节点结构介绍，可见 [Design](/workflow/design) 部分。

<Img src="/workflow/quickstart/3.png" />

## Input Data

接下来，我们需要输入实际运行的内容，点击每个节点中的输入部分，在弹出窗口中输入以下脚本内容：

<CodeGroup>

```bash {{ title: 'POSCAR' }}
# 晶格结构定义文件，包含晶格常数、基矢和原子坐标（示例为面心立方Si的笛卡尔坐标）
fcc:
   3.9
 0.5 0.5 0.0
 0.0 0.5 0.5
 0.5 0.0 0.5
   1
cartesian
0 0 0
```

```bash {{ title: 'INCAR' }}
# 核心控制文件，设置截断能240eV、高斯展宽0.1eV，初始化电荷计算
general:
 System = fcc Si
 ISTART = 0 
 ICHARG = 2
 ENCUT  = 240
 ISMEAR = 0 
 SIGMA = 0.1
```

```bash {{ title: 'KPOINTS' }}
# Monkhorst-Pack方法生成11×11×11 k点网格，用于布里渊区积分
K-Points
 0
Monkhorst Pack
 11 11 11
 0  0  0
```

</CodeGroup>

<Note> PROTIUM 支持多种不同的用户输入方式，但是对于结构灵活的可运行脚本，这里主要使用纯文本框 </Note>

## Running

输入完毕后，点击 <kbd>`运行`</kbd> 按钮开始运行，节点将显示黄色的运行中状态。

<Img src="/workflow/quickstart/4.png" />
